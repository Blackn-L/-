## 字符串的常用方法

``` JavaScript
let str = 'hello'
1、str.charAt(1) # 1为索引，查找对应索引位置字符
# 返回值：对应索引位置字符，无则为空字符串
2、str.charCodeAt() # 查找对应索引位置字符的Unicode 值
# 返回值：找到对应字符的索引的Unicode 值，无则返回NaN
3、str.indexOf() # 查找某个字符在字符串中的索引
# 返回值：对应字符的索引，无则返回-1
4、str.substring() # 从字符串截取一些字符
     str.substring(3,7) # 开始位置3，结束位置7，截取 3，4，5，6，包前不包后
# 返回值：截取的字符串，无则返回空字符串
# 任意参数小于0或NaN，则被当作0
# 省略indexEnd，则截取到最后
# 如果indexStart < indexEnd，则两者换个位置，如(7,3)变成(3,7)
5、str.substr() # 从字符串中截取一些字符(开始位置，截取个数)
#  返回截取的字符串，无则返回空字符串
6、str.toLowerCase() # 将字符串全部转为小写
# 返回值：小写后的字符串，不改变原字符串
7、str.toUpperCase() # 将字符串全部转为大写
8、str.split() # 将字符串以指定分隔符分割
# 返回值：数组，不改变原字符串，如果无分割则返回长度为1的数组['hello']
9、str.replace(('hel', '666')) # 返回666lo
# 返回值：替换后的字符串，不改变原字符串，无则返回原字符串，只替换一次
```
